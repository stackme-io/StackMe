# StackMe

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Free browser tools for data teams. No sign-up, no server, no uploads.

- **AnalyzeMe** — detect missing values, duplicates and outliers in CSV/JSON, entirely in your browser (DuckDB-Wasm)
- **ForgeMe** — generate synthetic datasets with injected anomalies for testing

**[stackme-app.vercel.app](https://stackme-app.vercel.app)**
