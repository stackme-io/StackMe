Inconsistencies to Resolve

opacity
ForgeMe Sidebar section labels use text-muted-foreground (100%) but AnalyzeSidebar uses text-muted-foreground/70. Recommend: unify to /70.
opacity
ForgeMe footer badges use text-muted-foreground/75, AnalyzeMe uses /70. Recommend: unify to /70.
color
Anomaly count in ResultBar uses text-amber-500, but anomaly type labels in AnomalyCards use text-amber-400. Recommend: standardize on amber-400 for outlier/count color.
duplication
RoadmapTab.tsx is duplicated in forge-me/tabs/ and analyze-me/tabs/ with identical structure. Candidate for extraction to shared/RoadmapTab.tsx with a namespace prop.
hardcoded
ForgeMe Sidebar section labels ("Anomaly mix", "Mode", "Hide upcoming", "+ N upcoming") are hardcoded English strings — not wired to i18n.
hardcoded
ResultBar "All rows", "⚠ Anomalies only", "Copy", "Copied", "Export", "Exported" are hardcoded. "All rows" is passed through t() but the rest are not.
